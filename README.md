An advanced e-commerce Flutter application designed to provide a seamless shopping experience.

### Authentication
- **Login, Sign Up, and Forgot Password:** Secure user authentication to access the app.

### User Profile
- **Gender and Age Selection:** Choose user profile details to customize the shopping experience.

### Product Browsing
- **Categories and Search:** Browse through different categories and use the search feature to find specific products.
- **Product Details:** View images, titles, descriptions, and choose product variations like size, color, and quantity.
- **Add to Cart:** Easily add products to the cart for checkout.

### Checkout Process
- **Location and Payment:** Enter delivery location and complete payment. Payment is dynamic and simulated for demonstration.

### Additional Features
- **Wishlist:** Save favorite products for quick access.
- **Support:** Contact support via WhatsApp or call for assistance.
